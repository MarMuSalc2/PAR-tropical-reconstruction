# PAR-tropical-reconstruction

Reproducible code and data for PAR (Photosynthetically Active Radiation) reconstruction at a humid tropical coastal site.

## Repository contents

`Methodology_PAR_M0_M5.Rmd` — R Markdown with the reconstruction methodology (models M0–M5).

`PAR_NSRDB2024_reconstruido.xlsx` — Dataset used for the PAR reconstruction.
